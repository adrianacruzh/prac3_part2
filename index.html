Etiquetas de CSS

@media

aquí es en donde se ponen las caracteristicas del dispositivo en donnde que van a estar ejecutando los sitios web..

  • entendido

box-sizing

Es una propiedad CSS para cambiar el modelo de caja por defecto de los naavegadores.El ancho de un elemento se altera si se le aplica un borde o un padding.

  • interesante

buttom

Permite desplazar un elemento respecto a su posición original tomando como referencia el borde inferior del elemento.

  • vaya

Padding

Establece el espacio de relleno requerido por todos los lados de un elemento. El área de padding es el espacio entre el contenido del elemento y su borde ( border ) no se permiten valores negativos.

  • excelente

margin

Es una de las "propiedades shorthand" que define CSS y que se utilizan para establecer de forma abreviada el valor de una o más propiedades individuales.

  • entendido
